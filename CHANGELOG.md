# Changelog

## [1.2.0] - 2025-12-07
### Added
- **Complete English Translations**: All missions, UI elements, planet descriptions, and quiz/XP systems are now available in English.
- **Mission Fixes**: Improved logic to prevent mission duplication and ensure missions autocomplete if the planet was already visited.
- **Debug Tools**: Added `resetMissions()` to console for debugging progression.
- **UI Polish**: Improved "Controls" HUD and XP Panel layout.

### Fixed
- Fixed bug where Mission 2 would display as Mission 1.
- Fixed crash when switching languages (removed page reload).
- Fixed HUD translation issues.

## [1.1.0] - 2025-12-06
### Added
- **Photo Mode**: Take screenshots with UI Overlay.
- **Planet Comparator**: Visual comparison tool.
- **Collectibles**: Hidden items to find.
- **PWA**: Offline support.
- **Time Controls**: Pause, speed up, slow down orbits.
- **Modern UI**: Toolbar, Mini-map, and clean HUD.

## [1.0.0] - 2025-12-05
### Added
- Initial 3D Solar System.
- Basic navigation and info panels.
- Portuguese content.
